## 1. API v1 端点测试

### TC-API-001 — v1 端点可访问性

| 项目 | 内容 |
|------|------|
| **优先级** | P0 |
| **描述** | 验证所有 13 个 v1 端点均可正常访问并返回正确 Content-Type |
| **前置条件** | 服务运行中，Redis 可用 |

**测试步骤：**

| # | 端点 | 方法 | 预期状态码 | 预期 Content-Type |
|---|------|------|-----------|-------------------|
| 1 | `/api/v1/today.json` | GET | 200 | `application/json; charset=utf-8` |
| 2 | `/api/v1/today.css` | GET | 200 | `text/css; charset=utf-8` |
| 3 | `/api/v1/index-data.json` | GET | 200 | `application/json; charset=utf-8` |
| 4 | `/api/v1/date=05-25` | GET | 200 | `application/json; charset=utf-8` |
| 5 | `/api/v1/theme/yozakura-reverie.json` | GET | 200 | `application/json` |
| 6 | `/api/v1/diy/themes.json?sort=new&size=5` | GET | 200 | `application/json` |
| 7 | `/api/v1/diy/theme.json?id=nonexistent` | GET | 404 | `application/json` |
| 8 | `/api/v1/diy/submit.json` | OPTIONS | 204 | — |
| 9 | `/api/v1/diy/like.json` | OPTIONS | 204 | — |
| 10 | `/api/v1/ai/generate.json` | OPTIONS | 204 | — |
| 11 | `/api/v1/admin/health.json` | GET | 200 | `application/json` |
| 12 | `/api/v1/admin/login.json` | OPTIONS | 204 | — |
| 13 | `/api/v1/admin/review.json` | GET | 401 | `application/json` |

**curl 命令：**

```bash
# 1. 今日主题
curl -s -o /dev/null -w "%{http_code}" "http://localhost:4399/api/v1/today.json"

# 2. CSS 端点

# 3. 索引数据

# 5. 预设主题
curl -s -o /dev/null -w "%{http_code}" "http://localhost:4399/api/v1/theme/yozakura-reverie.json"

# 6. 社区主题列表
curl -s -o /dev/null -w "%{http_code}" "http://localhost:4399/api/v1/diy/themes.json?sort=new&size=5"

# 7. 不存在的主题
curl -s -o /dev/null -w "%{http_code}" "http://localhost:4399/api/v1/diy/theme.json?id=nonexistent"
```

### TC-API-006 — 限流中间件验证

| 项目 | 内容 |
|------|------|
| **优先级** | P1 |
| **描述** | 验证 `/api/v1/diy/submit.json` 和 `/api/v1/diy/like.json` 的限流规则生效 |
| **前置条件** | 准备一个有效的 POST body |

**测试步骤：**
1. 向 `/api/v1/diy/submit.json` 连续发送 4 次 POST 请求（间隔 < 60s）
2. 第 4 次应返回 `429 Too Many Requests`
3. 等待 60s 后，再次发送应返回正常

```bash
for i in 1 2 3 4; do
  curl -s -o /dev/null -w "Request $i: %{http_code}\n" \
    -X POST "http://localhost:4399/api/v1/diy/submit.json" \
    -H "Content-Type: application/json" \
    -d '{"name":"Test","author":"T","cssVars":{"--color-primary":"#000","--color-bg":"#fff"}}'
done
# Request 4 预期: 429
```
