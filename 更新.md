## 一、API 版本化迁移 — 全面引入 `/v1/`

### 1.1 动机

ThemeDist 的核心用法是让用户在其 `<script>` 标签中直接硬编码 API 地址。一旦部署，服务端便失去对客户端代码的控制权。引入版本号控制后，可以在 `/v2/` 中放心地发布破坏性变更，而老用户的网站继续安稳运行在第一版协议上。

### 1.3 v1 端点变更详情

所有 13 个 API 端点文件移至 `src/pages/api/v1/`，相对导入路径全部修正（比原路径多一层 `../`）。

**所有 JSON 响应体新增 `apiVersion: 'v1'` 字段**，方便客户端检测协议版本。`today.css.ts` 返回 CSS 文本，不包含此字段。

### 1.4 旧端点处理

原有的 13 个非 v1 API 文件及其 4 个空子目录（`theme/`、`diy/`、`ai/`、`admin/`）已全部移除。旧路径 `/api/today` 通过 CDN 边缘层 301 重定向至 `/api/v1/today.json`。

### 1.5 基础设施配置更新

**`vercel.json`**

- 重定向：`/api/today` → `/api/v1/today.json` (301)
- 缓存头规则移至 v1 路径：`/api/v1/today.json`、`/api/v1/theme/(.*)`、`/api/v1/admin/(.*)`
- 保留通用兜底规则 `/api/(.*)` 覆盖所有 API 路径

**`netlify.toml`**

- 重定向：`/api/today` → `/api/v1/today.json` (301)
- 缓存头规则移至 v1 路径：`/api/v1/admin/*`、`/api/v1/theme/*`
- 保留通用兜底规则 `/api/*`

**`src/middleware.ts`**

- 限流规则只保留 v1 路径：`/api/v1/diy/submit.json` (3 次/分钟)、`/api/v1/diy/like.json` (10 次/分钟)

### 1.6 前端页面更新

## 二、Header 导航栏全面美化

### 2.1 导航栏容器

**滚动状态增强：**

```css
box-shadow:
  0 1px 0 0 color-mix(in srgb, var(--color-primary) 12%, transparent),
  0 4px 24px rgba(0,0,0,0.06);
```

滚动时顶部出现微妙的主题色发光线，底部新增柔和阴影，增强层次感。

### 2.2 Logo 图标（Orb）— 三层结构

| 层 | 实现方式 | 视觉效果 |
|----|----------|----------|
| 底色 | 圆锥渐变 `primary → secondary → accent` | 三色旋转色盘 |
| 轨道环 | `::before` 伪元素，2.5px 半透明边框 | 绕 orb 持续旋转，带微光晕 |
| 高光点 | `::after` 伪元素，6px 白色半透明 + 模糊 | 模拟玻璃材质反光 |

**悬停效果：** 旋转 180° + 放大至 1.35 倍 + 双层外发光（primary + secondary 双色辉光）

**点击效果：** 继续旋转 180°（累计 360°）+ 缩小回弹至 0.9 倍

**动画：**
- `orbBreath` — 4 秒呼吸缩放（1 → 1.06）
- `orbRing` — 3 秒轨道环持续旋转
- `orbHighlight` — 4 秒高光点明暗呼吸

### 2.3 Brand 文字

- 字重升级为 `800`
- 渐变色从 2 色扩展为 3 色（primary → secondary → accent），`background-size: 150%`
- 常驻 `drop-shadow` 微光晕
- 悬停 `.nav-home` 时：`letter-spacing` 从 `-0.03em` 展开至 `0.01em`，光晕增强，触发 1.5 秒渐变色位置扫过动画（`brandShimmer`）
- Logo 整体悬停放大 1.03 倍

### 2.4 Badge（主题名称标签）

- 新增 `::after` 流光动画 — 一道光晕每 3 秒从左到右扫过

### 2.5 导航链接

- 所有过渡从 `0.2s ease` 升级为 `0.3s cubic-bezier(0.25, 0.8, 0.25, 1.2)`
- **`::after` 下划线动画：** 从中心向外展开的渐变下划线，悬停时 `scaleX(0 → 1)`
- **悬停效果：** 上浮 1px + 环形发光边框 + 背景色加深
- **激活状态：** 下划线常驻显示

### 2.6 GitHub 图标

- 圆角升级为 `12px`，新增 `1px solid transparent` 边框
- **悬停效果：** 放大至 1.08 倍 + 边框淡入 + 外发光 + 图标旋转 -8°
- **点击效果：** 缩小至 0.94 倍 + 0.1s 快速回弹

### 2.7 Monitor 按钮 — 实时监控台入口

**位置：** 导航链接末尾，GitHub 图标之前
**链接：** `https://themedist-monitor.vercel.app/`

**设计元素：**

| 元素 | 效果 |
|------|------|
| 脉冲圆点 | 呼吸式缩放 + 双层光晕，强调"实时"感 |
| 雷达图标 | 虚线圆 + 扫过弧形 + 时钟指针，悬停时旋转 180° |
| 渐变文字 | accent → primary 渐变 |
| 扫描线 | `::before` 伪元素，4 秒一次从左到右扫过 |
| 悬停效果 | 双层外发光 + 内阴影 + 上浮 1.5px + 毛玻璃模糊 |
| 移动端 | 全宽显示，附带交错淡入动画 |

### 2.8 汉堡菜单（移动端）

- 边框淡入 + 颜色过渡 + 悬停放大 1.05 倍
- 菜单面板：向下滑动动画（`menuSlide`）
- 每个菜单项：从左淡入 + 依次延迟（`linkFadeIn`，相邻项间隔 0.04s）
- 增强的毛玻璃背景（`saturate(1.3)`）+ 阴影

## 三、首页全面重构

> 所有 CSS 类名使用 `hp-` 命名空间，避免与 Layout 全局样式冲突。

### 3.1 Hero 区域

**背景氛围：**
- 3 个浮动光球，各自独立 8 秒运动轨迹（`hpOrbFloat`），颜色分别取自 primary、secondary、accent

**状态标识：**
- 脉冲圆点动画（`hpPillPulse`，2.5 秒周期）
- 显示当前日期（JS 动态渲染）

**标题：**
- 响应式字号 `clamp(2.6rem, 7vw, 4.4rem)`，字重 800
- "独一无二" 使用三色渐变 + 6 秒背景位置扫过动画（`hpGradShift`）

**CTA 按钮：**
- 主按钮：实色背景 + `box-shadow` 彩色阴影 + 悬停上浮 2px
- 幽灵按钮：毛玻璃背景 + 悬停边框变色

**主题预览卡片：**
- 毛玻璃材质 + 微妙边框 + 半透明阴影
- 悬停时：上浮 4px + 增强阴影 + 色条比例微动 + 4 个色块各自旋转/缩放
- 卡片下方新增 4 个统计数字（内置主题数 / CSS 变量数 / 双端部署 / 正常运行时间）

### 3.2 安全状态面板

- CSS 类名重构为 `hp-sec-*`
- 悬停卡片微阴影
- 健康/警告/危险三色边框 + 圆点 + 标签联动

### 3.3 三步接入

- 数字被替换为语义化 SVG 图标（显示器 / 齿轮 / 时钟）
- 步骤之间插入箭头连接线（移动端隐藏）
- 悬停时卡片边框发光 + 图标放大 + 外光晕
- 移动端纵向堆叠

### 3.4 功能亮点

- 全部 6 个 emoji 替换为内联 SVG 图标：
  - 🎨 → 调色盘（同心圆）
  - 🔤 → 排版（段落符号）
  - 📐 → 间距（网格布局）
  - ✨ → 阴影（图层堆叠）
  - 🌿 → 节日（太阳）
  - 🖼️ → 壁纸（图片占位）
- 悬停时：上浮 3px + 边框渐变为主题色 + 图标背景加深 + 外发光

### 3.5 代码示例卡片

**新增复制按钮：**
- 每个代码块右上角有复制图标按钮
- 点击后切换为 ✓ 对勾图标 + 绿色高亮，1.8 秒后自动恢复
- 使用 `navigator.clipboard.writeText()` API

**视觉改进：**
- 终端圆点使用 `<i>` 元素（替代原来的 `<span>`）
- 毛玻璃悬停效果 + 边框高亮
- 代码块可滚动区域增大至 400px

### 3.6 CTA 区域

- 增强的 `linear-gradient` 背景（145deg，三色混合）
- 双层 `box-shadow`：外层大范围辉光 + 内层 inset 微光
- 实时显示预设主题数量
- 大号按钮（54px 高度 / 16px 字号 / 16px 圆角）

### 3.7 入场动画

- 所有 `reveal` 动画使用 `cubic-bezier(0.25, 0.8, 0.25, 1)` 缓动
- 5 级延迟梯度（0.1s / 0.2s / 0.35s / 0.5s / 0.65s）
- 从 `translateY(24px)` + `opacity: 0` → 原位 + `opacity: 1`
